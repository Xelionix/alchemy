About Alchemy Engine

Alchemy Engine is a game engine for text-based games, created by Xelionix. It comes with built-in multi-language support for Batch, PowerShell, Python, and ALC Script.

Engine Features

  • All-in-One User Interface
  • Dual-Terminal Display with Project Hierarchy
  • Complete and Polished Project Management System
  • Built-In Support for Dialogue, Branches, Inventories, and Variables
  • Save and Load Systems

System Modes

Batch Mode

Batch Mode allows for the widest range of compatibility using Alchemy Engine. It can support operating systems as far back as Windows 95 if programmed accordingly. This mode is best for simple UIs and an easy programming experience.

PowerShell/Batch Mode

The PowerShell/Batch Mode allows for easy programming but with built-in engine support for `.ps1` files, enabling greater customization on newer OSes.

EXE Modes

EXE Modes include Pure Python and Standard. Standard uses ALC Language for easily programming complex TUIs. Pure Python skips ALC and allows users to code directly in Python. Both modes can compile to EXE.
